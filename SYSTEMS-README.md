# AI 拓边实验室 - 系统说明

## 系统概述

本系统包含三个主要部分：
1. **报名系统** (`index.html`) - 研究主题报名
2. **讲演预约系统** (`presentation-booking-simple.html`) - 预约讲演时间
3. **讲演打分系统** (`scoring-system.html`) - 对讲演者进行评分

## 系统功能

### 1. 报名系统
- 20名GNSS算法研发部成员报名4个研究主题
- 每个主题限5人参与
- 每人只能选择一个主题
- 基于Supabase实时数据库

### 2. 讲演预约系统
- 从2026年4月23日开始，每周三进行讲演
- 每次3人讲演，共7周完成所有20人
- 每人只能预约一个时间段
- 实时显示预约状态

### 3. 讲演打分系统
- 剩余19人对讲演者进行打分（1-5分）
- 每人只能给同一个讲演者打一次分
- 不能给自己打分
- 自动计算平均分并排名

## 数据库表结构

### 1. 报名表 (registrations)
- `id` - 主键
- `theme_id` - 主题ID (1-4)
- `name` - 参与者姓名
- `created_at` - 创建时间

### 2. 讲演预约表 (presentation_bookings)
- `id` - 主键
- `participant_name` - 参与者姓名
- `week_id` - 周次 (1-7)
- `slot_number` - 讲演顺序 (1-3)
- `booked_at` - 预约时间

### 3. 讲演评分表 (presentation_scores)
- `id` - 主键
- `rater_name` - 评分人姓名
- `presenter_name` - 讲演者姓名
- `score` - 分数 (1-5)
- `comment` - 评价意见
- `scored_at` - 评分时间

## 使用流程

1. **报名阶段**：所有成员在报名系统中选择研究主题
2. **预约阶段**：报名成功后，在讲演预约系统中预约讲演时间
3. **讲演阶段**：按照预约时间进行讲演
4. **评分阶段**：其他成员在打分系统中对讲演者评分
5. **统计阶段**：系统自动计算平均分并排名
